Krisha.kz Real Estate Scraper \& Scorer

An automated tool designed to extract real estate listings and evaluate them based on a 1–10 value-for-money index.



Core Engine: XGBoost model trained with Krisha.kz’s price analyzer data.



Interface: PyQt GUI for intuitive data navigation (AI-assisted).



Documentation: Comprehensive technical docs included (AI-assisted).



Quick Start: Run app.py to launch the application.
